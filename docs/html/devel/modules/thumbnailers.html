5.16. thumbnailers — Utilities to generate a thumbnail from a file¶

This module contains utilities to generate a thumbnail from a file.

Thumbnailers are registered with ThumbnailersManager.

  • 5.16.1. thumbnailers.base — Base classes for thumbnailers
  • 5.16.2. thumbnailers.jfif — Thumbnailer for files containing a JPEG thumbnail
    • 5.16.2.1. How it works
  • 5.16.3. thumbnailers.magick — thumbnailer for image files (ImageMagick)
  • 5.16.4. thumbnailers.odf — Thumbnailer for OpenDocument files
  • 5.16.5. thumbnailers.pil — thumbnailer for image files
  • 5.16.6. thumbnailers.png — Thumbnailer for files containing a PNG thumbnail
    • 5.16.6.1. How it works
  • 5.16.7. thumbnailers.sw — Thumbnailer for SolidWorks files
