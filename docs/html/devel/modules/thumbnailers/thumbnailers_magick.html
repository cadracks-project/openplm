5.16.3. thumbnailers.magick — thumbnailer for image files (ImageMagick)¶

plmapp.thumbnailers.magickthumbnailer.limit_resources()[source]¶

Limits the process execution time to 60 seconds.

plmapp.thumbnailers.magickthumbnailer.magick_thumbnailer(input_path, original_filename, output_path)[source]¶

Thumbnailer that calls convert (from ImageMagick) to generate a thumbnail.
