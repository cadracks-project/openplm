2.3. Plugin for Thunderbird¶

2.3.1. Build and Installation¶

2.3.1.1. Get the sources¶

This plugin is available on the svn in the directory trunk/plugins/thunderbird.

2.3.1.2. Dependances¶

Of course, you need Thunderbird, this plugin has been test with the versions 3.0 and 3.1.

2.3.1.3. Build¶

Note

You can skip this step, download this file. Note that this file may not be up to date.

Just go in the plugins/thunderbird directory and run the command ./build.sh. This shoul create a file named openplm.xpi.

2.3.1.4. Installation¶

  1. Launch Thunderbird

  2. Launch the add-ons manager: menu Tools ‣ Add-ons. A dialog should appear:

  3. Then click on the Install... button and select the file openplm.oxt. A dialog asking confirmation should appear:

  4. Click on the Install Now.

  5. Restart Thunderbird

  6. Now, the plugin is installed. If the plugin is installed, a new submenu named OpenPLM should be present in the File menu.

2.3.2. Usage¶

2.3.2.1. Configuration¶

First, you should specify where the server is located:

  1. Open the dialog (menu Tools – Add-ons).

  2. Select the OpenPLM add-ons:

  3. Click on the Preferences button. This dialog should appear:

  4. Enter your server’s location and close the dialog.

2.3.2.2. Login¶

Before checking-in a file, you sould login. Open the configuration dialog (menu File ‣ OpenPLM ‣ Login). This dialog should appear:

Enter your username and your password and click on Ok.

2.3.2.3. Check-in a mail¶

You can save a mail on the server:

  1. Select one or several mails

  2. Click on File ‣ OpenPLM ‣ Check-in current mail. This dialog should appear:

  3. Fill the search form and click on the Search button.

  4. Select your document, and click on Ok

  5. Your mail has been had.

2.3.2.4. Create a new document¶

You can create a new document from a mail:

  1. Select one or several mails

  2. Click on File ‣ OpenPLM ‣ Create a new document. This dialog should appear:

  3. Fill the form

  4. Click on Ok to validate the creation.

  5. Your document has been created.
